## ✅ Parte 1: Lógica de Programação com Dart

1. Contagem de pares
   Escreva uma função em Dart que receba um números inteiros e retorne quantos deles são pares.

2. Inverter uma string
   Crie uma função que receba uma string e retorne ela invertida (ex: "nathan" → "nahtan").

3. Fatorial
   Crie uma função que calcule o fatorial de um número inteiro n.

4. Soma dos dígitos
   Crie uma função que receba um número inteiro e retorne a soma dos seus dígitos.
   Exemplo: 123 → 6 (1+2+3)

5. Palíndromo
   Crie uma função que receba uma palavra e diga se ela é um palíndromo (lê-se igual de trás para frente, como "ana", "ovo").
